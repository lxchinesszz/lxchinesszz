### 什么是微服务？

微服务是一种现代的软件方法，应用程序代码以小的、可管理的部分交付，独立于其他部分。

### 为什么要构建微服务？

它们的小规模和相对隔离可以带来许多额外的好处，例如更容易维护、提高生产力、更大的容错性、更好的业务一致性等等。
