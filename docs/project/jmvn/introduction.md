目标打造一个简单好用的 Java 服务脚手架工具，让任何没有开发经验的同学都能快速构建一个可以运行的项目

工具围绕几个方面进行开发, 力求在以下方面提高开发的工作效率, 同时通过脚手架的方式从一开始就统一下项目结构和模型。

- 深度定制 (定制公司标准的项目结构)
- 快速构建标准的Maven多模块服务
- 支持数据库模型快速安装 (一次配置终生使用, 多快好省)
- 数据库模型文档一键导出 (每次表结构变更, 快速导出最新表文档)



## 介绍

快速构建支持SpringBoot的Maven多模块应用,启动及快速打包部署

- 支持Maven多模块构建、
- 数据库模型安装、
- 数据库模型导出),
- 统一项目规范,
- 提高工作效率,
- 可进行深度定制。
